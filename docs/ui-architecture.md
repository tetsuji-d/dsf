# DSF Studio Pro - UI Menu & Feature Architecture (Universal Design)

本ドキュメントは、ユニバーサルデザイン（UD）化に向けて再設計・整理された「エディタ（PC）」および「ビューワー（Mobile/PC）」のUI構造と機能のグループ分けを定義します。

## 1. 課題と整理の方針
現在のUIは機能追加の歴史により、リボンメニューや右サイドパネルに機能が散在している状態です（例：画像追加がリボンと右パネルの両方にある等）。これを以下の方針で整理します。

1. **機能の重複排除**: 同じ機能へのアクセスパスを一つに絞り、ユーザーの迷いを無くす。
2. **段階的開示（Progressive Disclosure）**: 常に全てを見せるのではなく、現在選択している要素（ページ、セクション、テキスト）に対して「今使える機能」だけを文脈に合わせて表示する。
3. **明確なグループ化**: 「全体に関わる操作（プロジェクト管理）」と「個別の編集操作（挿入・調整）」を物理的（左/右、上/下など）に分離する。

---

## 2. エディタ画面（PC向け）のUI構造

画面を大きく3つの役割に分割します。

### A. グローバルトップバー（画面最上部）
プロジェクト全体に関わる「メタ操作」と「状態確認」を集約します。

*   **左側 (Project Info)**
    *   🏠 ホームへ戻る（プロジェクト一覧モーダルを開く）
    *   作品タイトル入力欄（大きく見やすく）
    *   クラウド保存ステータス（「未保存」/「保存中...」/「自動保存済み」）
*   **右側 (Global Actions)**
    *   ▶ プレビュー（別タブでビューワーを開く）
    *   ⬇ エクスポート（`.dsf` 配信データ / `.dsp` バックアップデータ の選択プルダウン）
    *   👤 アカウント / ログイン状態

### B. 左サイドバー (Pages & Structure)
「全体構成」を管理するエリアです。これまで上部リボンにあった画面構成系の機能をここに集約します。

*   **タブ 1: Pages (サムネイル一覧)**
    *   機能: ページの並び替え (D&D)、ページ複製、ページ削除
    *   ツール: 列数変更 (1〜8列切替)
*   **タブ 2: Structure (目次・構造)** *※新規追加*
    *   機能: 章・節・項の階層構造をツリー表示し、アウトラインを管理する。

### C. 中央キャンバス ＆ 右サイドバー (Contextual Edit)
中央で選択した「要素」に応じて、右サイドバーの内容を**動的に切り替え**ます。これにより、無駄なUIを隠して操作領域（ボタンサイズ）を大きく確保します。

*   **何も選択していない時 (Document Setup)**
    *   全体言語設定 (ja/en切替)
    *   横書き/縦書きのデフォルト設定
    *   デフォルトフォント設定
*   **「ページ（画像）」を選択した時 (Image Edit)**
    *   画像変更ボタン (大きく)
    *   画像位置・ズーム調整モードのトグル
    *   💬 吹き出しを追加するボタン
*   **「吹き出し / テキスト」を選択した時 (Text Edit)**
    *   テキスト入力エリア（大きく）
    *   リッチテキスト装飾ツールバー（B, I, 見出し, 左揃え等のボタンを大きく配置）
    *   吹き出しの形状変更プルダウン（💭/💥/💬）
    *   🗑 削除ボタン（危険なアクションは最下部に赤色で）

## 3. ビューワー画面（モバイル＆PC向け）のUI構造

「読書体験の邪魔をしない」ことと「操作の迷いがない」ことを両立させます。

### A. 読書中の画面（UI非表示時）
*   **左端 30% エリア**: 前のページへ戻る（不可視の広大なボタン）
*   **右端 30% エリア**: 次のページへ進む（不可視の広大なボタン）
*   **中央 40% エリア**: タップで「ナビゲーションUI」を表示/非表示トグル

### B. ナビゲーションUI（中央タップ時のみ表示）
半透明のオーバーレイとして、上下からアニメーションでスライドインします。

*   **上部ヘッダー**
    *   `[閉じる ✕]` ボタン（左上）
    *   作品タイトル（中央）
    *   `[言語切替]` または `[共有 🔗]` ボタン（右上）
*   **下部フッター（操作エリア）**
    *   **ページスライダー**: ページ全体を一気に飛ばせる大きなスライダーバー。ドラッグ中は「現在のページ数」がツールチップで大きく表示される。
    *   **ページナビゲーション**: スライダーの両脇に、明確な `[◀ 戻る]` と `[進む ▶]` のボタン（最低44px角を確保）。
