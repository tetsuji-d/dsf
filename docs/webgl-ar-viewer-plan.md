# WebGL AR ビューワー実装計画 — ⚠️ 廃案

**ステータス**: **廃案（2026-03-25）**

## 廃止理由

- 実装が複雑すぎる
- 端末ごとに安定した表示を実現するのが困難
- 動作が重くなりがちでUXが低下する

## 代替方針（Gen 3）

**WebP 画像のみ**でページを構成するシンプルな方式を採用。

- すべてのページは WebP 画像として扱う
- ビューワーは WebP 画像の高速表示に専念
- Three.js / WebGL / WebXR は使用しない

詳細は `AGENTS.md` のフォーマット方針セクションを参照。
