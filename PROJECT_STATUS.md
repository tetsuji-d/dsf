# PROJECT_STATUS

## 目的
Webtoonスタイルの縦スクロールコミックを作成できるWebエディタ「DSF Studio Pro」およびモバイル向け閲覧ビューワー。

## 現状
- **ステータス**: 稼働中 (ローカル / Firebase Hosting)
- **本番URL**: `https://vmnn-26345.web.app`
- **開発URL (LAN例)**: `http://192.168.2.102:5173/`
- **利用想定**: PC(エディタ) / Mobile(ビューワー)

## 実装済み主要機能
- Firebase Authentication (Google) によるログイン/ログアウト
- Firestore 保存/読込、Storage 画像アップロード、共有URL生成
- ビューワー側のサインイン導線追加（未ログイン時のエラー案内含む）
- Pagesパネル改善:
  - サムネイル列数 `8/5/4/2/1` 切替
  - プロジェクト単位 + デバイス別(PC/モバイル)の列数保存
  - アクティブサムネイルの強調表示
  - サムネイルD&D並び替え（PC）/ 長押し並び替え（モバイル）
  - サムネイル間の `+` 挿入 / ページ複製
- テキストセクション改善:
  - 固定組版モジュール (`js/layout.js`) 導入
  - 言語別の組版制約（例: 日本語縦12行、英語横21行）
  - overflow検知と「次ページへ分割」機能
  - エディタ/ビューワーの表示差異縮小
  - 日本語縦書き時のビューワーナビ方向連動（左右ボタン/スライダー）
- UI改善:
  - Editパネル（右サイドバー）のリサイズ機能追加
  - 作品タイトル入力を Edit パネルからヘッダー中央へ移動
  - モバイルドロワー閉じる挙動の修正
- Local-First アーキテクチャ強化:
  - 認証なしのGuestモード（ログイン不要）でのエディタ/ビューワー利用対応
  - `.dsp` (プロジェクト圧縮データ) のエクスポート・ローカルインポート実装
  - `.dsf` (配信コンテンツデータ) のエクスポート実装、およびビューワーのオフライン表示対応
  - 保存・エクスポート時の必須項目緩和（警告を無視しての強制書き出し対応）

## ローカル起動手順
```bash
npm install
npm run dev
```

## デプロイ手順
```bash
# ビルド
npm run build

# Firebaseへのデプロイ
npx firebase deploy --only hosting
```
